# 🎨 Frontend – UniSalas

O frontend do **UniSalas** é uma aplicação web desenvolvida em **Angular**, responsável pela interface do usuário e consumo da API REST.  

---

## 🛠️ Tecnologias Utilizadas

- Angular
- TypeScript
- HTML5
- CSS / SCSS

---

## 📋 Pré-requisitos

Antes de iniciar, certifique-se de ter instalado em sua máquina:

- Node.js (recomendado: versão 18 ou superior)
- npm (gerenciador de pacotes)
- Angular CLI

Instale o Angular CLI globalmente:

```bash
npm install -g @angular/cli
```

---

## ▶️ Como Executar o Projeto

1. Acesse a pasta do frontend:

```bash
cd frontend
```

2. Instale as dependências:

```bash
npm install
```

3. Execute a aplicação:

```bash
ng serve
```

4. Acesse no navegador:

```
http://localhost:4200
```

---

## 🔗 Integração com Backend

O frontend consome a API do UniSalas.

Certifique-se de que o backend esteja em execução em:

```
http://localhost:8085
```

---

## 📌 Observações

- Certifique-se de que a API esteja ativa antes de utilizar o sistema completo.
- Caso ocorra conflito de porta, utilize:

```bash
ng serve --port 4201
```
